R Brette has implemented a Brian 2 version of the Destexhe et al 1998 model:

Destexhe A, Neubig M, Ulrich D, Huguenard J (1998) Dendritic low-threshold calcium currents in thalamic relay cells. J Neurosci 18:3574-88
doi: 10.1523/JNEUROSCI.18-10-03574

The model is available at the Brian 2 website where it can be run and edited without installing any software (in a Binder service):

https://brian2.readthedocs.io/en/stable/examples/frompapers.Destexhe_et_al_1998.html
